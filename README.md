AeroLink is an intuitive, all-in-one web portal designed to streamline international flight discovery and seat reservations into a single, high-contrast dashboard.

# ✈️ AeroLink

> **International Flight Operations & Reservation Dashboard**

A modern, high-contrast Web UI for searching and reserving international flights. Built with clean HTML5, CSS3, and JavaScript.

## 🌟 Key Features

* 🎯 **Split-Pane Layout** — Fixed booking controller alongside a scrollable flight schedule.
* ⚡ **One-Click Flight Selection** — Auto-populates selected flight data directly into the reservation summary.
* 🔄 **Animated Portal Loader** — Clean JavaScript lifecycle overlay with full-screen dark theme.
* ✅ **Interactive Validation** — In-browser form processing with real-time green success feedback.
